API Usage

Base endpoint: /api/reserves/ - Use query parameters to trigger scenarios.

?scenario= Named scenarios: under_limit, exceeds_limit, connection_timeout, invalid_json, etc.
?delay= Custom delay in milliseconds (e.g., ?delay=5000 for 5 seconds)
?error= HTTP error codes: 400, 401, 403, 404, 429, 500, 502, 503, 504
?size= Custom response size in KB (e.g., ?size=200 for 200 KB)
